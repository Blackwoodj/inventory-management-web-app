# Inventory Management Web Application

## Overview
This project is a web application that allows users to register, log in, and manage stock inventory.

The system supports adding, updating, and removing stock items, simulating a hospitality management system.

## Features
- Login system
- Add, update, and delete stock items
- Data storage and retrieval
- Basic UI for interacting with inventory

## Technologies Used
- PHP, SQL, JavaScript, HTML, CSS


## Purpose and Learning
- This application was developed early on into my web application learning experience, but was the culmination of various systems being incorporated into a single application to work in unison.
- The main challenges of this application were designing with PHP in mind, along with integrating with session data.
